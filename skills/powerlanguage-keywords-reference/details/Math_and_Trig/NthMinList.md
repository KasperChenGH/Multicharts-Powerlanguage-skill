# NthMinList

**Category:** Math_and_Trig
**Signature:** `NthMinList(N,Value1,Value2,Value3, etc.)`

Yields the Nth lowest value of the specified numerical expressions.

**Example (illustrative)**
```
Value1 = NthMinList;
```
