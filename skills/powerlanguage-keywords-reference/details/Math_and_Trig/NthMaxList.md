# NthMaxList

**Category:** Math_and_Trig
**Signature:** `NthMaxList(N,Value1,Value2,Value3, etc.)`

Yields the Nth highest value of the specified numerical expressions.

**Example (illustrative)**
```
Value1 = NthMaxList;
```
